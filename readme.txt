== Installation ==

1. Extract zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Requires CURL extension to be active in php configuration. (Just try the plugin and you'll know)
4. Requires PHP5.

Steps to take after plugin is installed:

A more detailed setup guide available here: http://tinyurl.com/wp-hyves-setup-guide

1. Go to WP-Hyves under Settings in WP-Admin and click on the link to apply for API Consumer Key and Secret.
2. Enter  Details such as:
- Your username.
- Contact email.
- Abuse email(usually same as blog email).
3. Click on the "Create a new Desktop consumer" link.
4. Enter details about your blog. The important part is the IP Address: just copy the ip-address you'll find in the WP-Hyves Settings page.
5. Now that the Consumer Key and Secret are created, copy them to your WP-Hyves settings.
6. Or open wp-hyves.php en fill in the blank config variables for setting the key and secret globally.

Now save changes and you're all done!
